# Agent Memory

This folder stores repo-local working memory for future agents. It is intentionally checked into the private repository so implementation decisions survive context resets.

Rules:

- Keep entries short and factual.
- Record durable product/architecture decisions, not transient TODOs.
- Do not store secrets, credentials, private phone numbers, or raw user data.
- Update `hana-chat.md` when a change affects future implementation choices.
